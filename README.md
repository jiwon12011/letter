# Letter - 편지를 보내세요

## 사용자 흐름 (User Flow)

```
┌─────────────────────────────────────────────────────────────────┐
│                        🏠 랜딩 페이지                            │
│                                                                 │
│   "소중한 사람에게 마음을 전하세요"                                  │
│                                                                 │
│   [ 편지 만들기 ]                                                │
└────────┬────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────┐
│              ✏️ STEP 1. 편지 쓰기                                │
│                                                                 │
│   받는 사람:  [ 홍길동      ]                                     │
│   편지 내용:  ┌─────────────────────────┐                        │
│              │ 자유롭게 작성...          │                        │
│              └─────────────────────────┘                        │
│   보내는 사람: [ 김철수      ]                                     │
│                                                                 │
│                              [ 다음 → ]                          │
└────────┬────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────┐
│              🎨 STEP 2. 꾸미기                                   │
│                                                                 │
│   봉투 색상:   🔴 🟤 🔵 🟢 🟣                                    │
│   편지지 스타일: [크림] [줄무늬] [도트] [꽃무늬]                     │
│   실링 모양:    ❤️  ⭐  🌙  🌸                                    │
│   폰트:        [손글씨1] [손글씨2] [정자체]                         │
│                                                                 │
│   ┌─ 실시간 미리보기 ─┐                                          │
│   │   봉투 미니 프리뷰  │                                         │
│   └──────────────────┘                                          │
│                                                                 │
│          [ ← 이전 ]  [ 다음 → ]                                  │
└────────┬────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────┐
│              📤 STEP 3. 미리보기 & 보내기                         │
│                                                                 │
│   ┌──── 최종 미리보기 ────┐                                      │
│   │  봉투 애니메이션 재생   │                                     │
│   │  → 실링 깨짐           │                                     │
│   │  → 플랩 열림           │                                     │
│   │  → 편지 등장           │                                     │
│   └──────────────────────┘                                      │
│                                                                 │
│   [ 🔗 링크 복사하기 ]  [ 다시 미리보기 ]                          │
│                                                                 │
│          [ ← 이전 ]  [ 새 편지 만들기 ]                           │
└────────┬────────────────────────────────────────────────────────┘
         │
         │  링크 공유
         ▼
┌─────────────────────────────────────────────────────────────────┐
│              💌 받는 사람 화면                                     │
│                                                                 │
│   "홍길동님에게 편지가 도착했습니다"                                 │
│                                                                 │
│   ┌────────────────┐                                            │
│   │  봉투 (닫힘)    │  ← 클릭하면 열림 애니메이션                   │
│   │  왁스 실링      │                                            │
│   └────────────────┘                                            │
│                                                                 │
│   → 클릭 → 실링 깨짐 → 플랩 열림 → 편지 슬라이드                   │
│   → 전체 화면 편지 읽기                                           │
│                                                                 │
│   [ 나도 답장하기 ]                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 설계 결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 데이터 저장 | URL hash (Base64 인코딩) | 백엔드 없이 링크 하나로 편지 전달 |
| 라우팅 | Hash-based SPA | 단일 HTML, 새로고침에도 상태 유지 |
| 스타일 커스텀 | CSS 변수 | 테마 교체가 간편 |

## 파일 구조

```
letter/
├── index.html          # SPA 메인
├── css/
│   └── style.css       # 전체 스타일
├── js/
│   └── main.js         # 라우팅 + 인터랙션
└── images/
```
